🎓 ARCHITECTURAL HIGHLIGHTS

  1. Repository Pattern
     Benefit: Data access encapsulation
     Future: Easy Firestore → API migration

  2. Riverpod Providers
     Benefit: Reactive state with auto-caching
     Future: Automatic re-fetching on invalidation

  3. CustomPaint vs SVG
     Benefit: 10ms render vs 100ms for complex transforms
     Tradeoff: Custom coordinate mapping required

  4. Set<String> Aggregation
     Benefit: O(1) lookups, automatic deduplication
     Tradeoff: Unordered (not an issue for glow display)

  5. Double Precision
     Benefit: Prevents Firestore int64 truncation
     Cost: 8 bytes vs 4 bytes per numeric field
