  🔴 PROBLEM: UI Displaying "Unknown"
═══════════════════════════════════════════════════════════════════════════════════

  WHAT USER SAW:
  ┌─────────────────────────────────────┐
  │ 🥩 Fuentes de Proteína              │
  │                                     │
  │  ⭕ Unknown (Sin descripción)       │
  │  ⭕ Unknown (Sin descripción)       │
  │  ⭕ Unknown (Sin descripción)       │
  │                                     │
  │ ❌ Food dropdowns completely broken │
  └─────────────────────────────────────┘

  ROOT CAUSE:
  ❌ Firestore documents have 4-NODE STRUCTURE:
     {
       "metadata": { "name": "Sardinas (Atlánticas)", ... },
       "content": { "calories": 208, ... },
       "app_integration": { "tip": "Omega-3...", ... },
       "quiz": { "impact": "Positivo", ... }
     }

  ❌ FoodModel.fromFirestore() was reading from ROOT LEVEL:
     final String name = data['name'] ?? 'Unknown';
     final String tip = data['tip'] ?? 'Sin descripción';

  ❌ Result: All fields not found → defaults to "Unknown"


═══════════════════════════════════════════════════════════════════════════════════
  ✅ SOLUTION: Multi-Node Aware Mapping
═══════════════════════════════════════════════════════════════════════════════════

  FILE: food_model.dart (Lines 167-256)
  METHOD: FoodModel.fromFirestore()

  UPDATED LOGIC:

  1️⃣  EXTRACT 4-NODE STRUCTURE:
      final metadata = data['metadata'] ?? {};
      final content = data['content'] ?? {};
      final appIntegration = data['app_integration'] ?? {};
      final quiz = data['quiz'] ?? {};

  2️⃣  MULTI-LEVEL FALLBACK (tries 4-node FIRST, then root):
      // ✅ Try metadata.name → Try root name → Default 'Unknown'
      final name = (metadata['name'] ?? 
                   data['name'] ?? 
                   'Unknown');

  3️⃣  EXTRACT EACH FIELD FROM CORRECT NODE:
      ┌──────────────────────────────────────────────────────┐
      │ Field        │ Source Node        │ Extracted Value   │
      ├──────────────────────────────────────────────────────┤
      │ name         │ metadata.name      │ "Sardinas..."     │
      │ category     │ metadata.category  │ "Proteinas"       │
      │ calories     │ content.calories   │ 208.0             │
      │ proteins     │ content.proteins   │ 22.0              │
      │ tip          │ app_integration... │ "Omega-3..."      │
      │ imr_score    │ app_integration... │ 10                │
      │ level        │ quiz.level         │ "Excelente"→ 3    │
      └──────────────────────────────────────────────────────┘

  4️⃣  HANDLE SPECIAL CASES:
      ✅ Serving can be "100g" (string) or 100 (number)
      ✅ Level can be "Excelente" (string) or 3 (number)
      ✅ Search tags in metadata as "tags" not "searchTags"
      ✅ Debug logging to verify extraction
